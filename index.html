⚙ SETTINGS

Beginner mode
Hides advanced features like candlesticks, Bollinger Bands, and the heat map. Plain English labels and bigger fonts. Switch off when you're comfortable.
Show price-change flash
Briefly highlight a row in green/red when its price updates. Off for a calmer screen.
Glossary tooltips
Hover any technical term (SMA, RSI, P/E, OHLC…) to see a plain-language explanation.
Theme
Pick a color palette. Default = neutral dark, Amber = classic Bloomberg orange-on-black, Green = vintage terminal, Light = bright daylight mode.
Sound cues
Audio beeps when an alert fires. Bloomberg-style chirp. Off for silent work.
Compact mode
Hide the scrolling ticker and sector strip for a denser chart-focused layout.
Language / Dil
Switch top-level interface labels. News + symbols stay in English (source data is English).
Restart welcome tour
Re-show the 5-step intro on the next load.
